# 💱 Currency Converter

A sleek currency converter with support for 12 major currencies.

## Features
- Convert between 12 currencies: USD, EUR, GBP, INR, JPY, CAD, AUD, CNY, SGD, AED, CHF, MYR
- Swap currencies with one click
- Shows inverse rate
- Quick-select popular currency pairs
- Flag icons for each currency
- Clean gradient result display

## Built With
- HTML5
- CSS3
- Vanilla JavaScript

## How to Run
Clone the repo and open `index.html` in your browser.
